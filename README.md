# Zed configuration

This repository stores my Zed Editor configuration and related local tooling.

## What’s here

- `settings.json` — editor preferences, UI settings, languages, and agent configuration.
- `keymap.json` — custom key bindings and Vim-mode shortcuts.
- `tasks.json` — reusable Zed tasks.
- `AGENTS.md` — global agent instructions for Zed.

## Notes

These files are meant to be synced with the Zed config directory and adjusted over time as my workflow changes.
